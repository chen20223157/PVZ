# 图片上传快速指南 ⚡

## 🎯 只需 3 步！

### 第 1 步：打开界面
按键盘 **`I`** 键

### 第 2 步：点击卡片
找到你想上传图片的植物/僵尸卡片，点击 **"上传图片"** 按钮

### 第 3 步：选择图片
选择你准备好的图片文件（PNG 或 JPG）

**完成！** 🎉 图片立即生效！

## 💡 常见操作

### 上传豌豆射手图片
1. 按 `I`
2. 找到 🌱豌豆射手 卡片
3. 点击"上传图片"
4. 选择图片
5. 完成！

### 切换到僵尸
1. 按 `I` 打开界面
2. 点击顶部 **"僵尸图片"** 标签
3. 看到所有僵尸类型
4. 点击卡片上传

### 删除图片
1. 找到已有图片的卡片（绿色）
2. 点击"删除图片"
3. 确认删除
4. 完成！

## ❓ 常见问题

### Q1: 图片没有显示？
**A:** 检查：
- 图片格式是否为 PNG/JPG
- 图片是否损坏
- 刷新页面重试

### Q2: 按钮是灰色的？
**A:** 已上传图片，按钮会变红色显示"删除图片"

### Q3: 找不到某个植物？
**A:** 使用滚轮滚动查看所有类型

### Q4: 图片太大上传失败？
**A:** 使用图片压缩工具，建议 <500KB
