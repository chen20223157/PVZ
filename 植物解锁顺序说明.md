## 🎮 新游戏流程

### 第1关 - 豌豆射手入门
```
初始阳光: 200
可用植物: 豌豆射手（100阳光/个）
僵尸数量: 10个普通僵尸

策略:
1. 种植2个豌豆射手（剩余0阳光）
2. 等待天降阳光（每8秒一个）
3. 继续种植豌豆射手
4. 小推车作为最后防线
```

### 第2关 - 向日葵经济
```
初始阳光: 150
可用植物: 豌豆射手 + 向日葵
僵尸数量: 15个普通僵尸

策略:
1. 先种1个豌豆射手防守（剩余50阳光）
2. 种1个向日葵发展经济（0阳光）
3. 继续平衡经济和防御
4. 后排向日葵，前排豌豆射手
```

### 第3关 - 坚果防御
```
初始阳光: 150
可用植物: 豌豆射手 + 向日葵 + 坚果墙
僵尸数量: 20个（普通 + 路障）

策略:
1. 后排：向日葵（经济）
2. 中排：豌豆射手（输出）
3. 前排：坚果墙（防御）
4. 三重防线，稳扎稳打
```

## 💡 设计理由

### 为什么第1关解锁豌豆射手？

#### 1. **玩家体验优先**
- ✅ 立即拥有攻击手段
- ✅ 可以主动防守
- ✅ 减少挫败感
- ✅ 增加游戏乐趣

#### 2. **游戏机制合理**
- ✅ 豌豆射手是最基础的防御植物
- ✅ 初始阳光200足够种2个
- ✅ 天降阳光可以持续补充
- ✅ 小推车提供安全保障

#### 3. **学习曲线更好**
- ✅ 第1关：学习攻击机制
- ✅ 第2关：学习经济系统
- ✅ 第3关：学习防御配合
- ✅ 渐进式教学

#### 4. **符合直觉**
- ✅ 塔防游戏首先需要防御
- ✅ 向日葵是辅助，不是主力
- ✅ 先攻后守更符合游戏逻辑
